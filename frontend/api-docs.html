Flux typique — tester l'API

1
Ouvre Auth Service → POST /api/auth/register → copie le token
2
Dans chaque Swagger, clique Authorize 🔒 → colle le token
3
Ouvre Game Service → GET /api/games/session?difficulty=EASY → 3 jeux retournés
4
Ouvre User Service → POST /api/users/results → soumets un score
5
Ouvre Leaderboard → GET /api/leaderboard/global → vois ton score dans le classement
